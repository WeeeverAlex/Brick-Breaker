Brick Breaker

O jogador deve esmagar uma parede de tijolos desviando uma bola quicando com uma raquete.
A raquete pode se mover horizontalmente e é controlada com as teclas "A" e "D".

Clique no botão abaixo para jogar

BRICK BREAKER


Sobre o jogo

O jogador possui 3 vidas
Os bricks também possuem vidas e conforme são atingidos suas cores mudam até serem quebrados por total
Conforme os bricks são quebrados, poderes são desbloqueados
A música muda de acordo com a pontuação
Para ganhar o jogo quebre todos os bricks


Poderes

  • Aumentar o paddle
  • Aumentar o número de vida
  • Aumentar velocidade da bola
  • Criar tiros na base

Como executar o jogo

  • Ao clicar no link você será direcionado a pagina de nosso drive
  • No drive siga a sequência de pastas Jogo > Pygame BrickBreaker
  • Baixe a pasta Pygame BrickBreaker(como todos os seus arquivos)
  • Coloque a pasta Pygame BrickBreaker no seus arquivos, na pasta do Visual Studio Code
  • Dentro do Visual Studio Code, rode o arquivo Menu.py
  • Após essas etapas você está pronto para jogar! Agora so se divertir!

Heuristicas

  • HEURÍSTICA 1: Os botões localizados na parte de cima do site não estão funcionando, dificultando a navegação do usuário pelo site
  • HEURÍSTICA 2: São usados alguns termos ('x1', 'ballbraker') que, aquelas pessoas que não sejam tão íntimas com a área de informática/jogos, talvez ficariam confusas
  • HEURÍSTICA 3: Como grande parte dos botões não estão funcionando no momento, não existe a possibilidade de consertar erros ou desfazer alguma ação
  • HEURÍSTICA 4: Não existem problemas presentes
  • HEURÍSTICA 5: Como grande parte dos botões não estão funcionando no momento, não existe a possibilidade de desfazer erros ou previni-los
  • HEURÍSTICA 6: Não existem problemas presentes
  • HEURÍSTICA 7: Os atalhos presentes na parte de cima da página não funcioinam, comprometendo a experiência do usuário
  • HEURÍSTICA 8: A densidade dos textos, principalmente na parte ('Sobre o jogo'), acabam causando um certo desconforto visual para o usuário
  • HEURÍSTICA 9: Não existe nenhuma mensagem de erro explicativa ao clicar algum dos botões que não funcionam (download,etc.)
  • HEURÍSTICA 10: No site, não há nenhum tipo de instrução de como abrir e jogar o jogo disponibilizado pelos criadores
